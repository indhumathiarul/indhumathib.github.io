How to add new guides

Step-by-step from your phone 📱

  1. Save your new .html guide file
  2. Go to your GitHub repo → guides/ folder
  3. Tap Add file → Upload files
  4. Upload your .html file — note the exact filename
  5. Open index.html in GitHub → tap the ✏️ edit icon
  6. Find the TOPICS config section (it says EDIT THIS)
  7. Add a new line: {'{'} title: "Your Guide Title", file: "guides/your-file.html" {'}'}
  8. Tap Commit changes → live in 2 minutes ✅

Security info

How your login works (no hardcoded password 🔒)

  1. Go to Firebase Console → Authentication → Sign-in method
  2. Enable Email/Password provider
  3. Go to Users tab → Add user
  4. Enter your email + a strong password
  5. That's it — your password lives only in Firebase, never in code
  6. To change password: Firebase Console → Authentication → Users → Reset
